Aprende todo lo necesario sobre Ruby y el desarrollo web


Ruby es un lenguaje que ha agarrado popularidad en los últimos méses es de crucial importancia que se enseñe en las aulas hoy en día porque cada vez más hay programadores y compañias que están dispuestas a proveer sus servicios usando lenguajes de programacion no tan convencioanles como es el caso de Ruby.


Sinatra es DSL para crear rápidamente aplicaciones web en Ruby con un esfuerzo mínimo. AWS Lambda es un manejador de funciones web que pueden desempeñar un sinfin de aplicaciones y triggers.





Qué es el desarrollo web?



Desarrollo web es un término que define la creación de sitios web para Internet o una intranet. Para conseguirlo se hace uso de tecnologías de software del lado del servidor y del cliente que involucran una combinación de procesos de base de datos con el uso de un navegador web a fin de realizar determinadas tareas o mostrar información.
